Traffic Simulator (Simple 2D)
Files:
- editor.py       : Tkinter map editor. Run `python editor.py` to create maps.
- map_utils.py    : Helper functions to load and build graph for simulator.
- simulator.py    : Pygame simulator. Run `python simulator.py sample_map.json`
- sample_map.json : A ready-to-run sample map.

Instructions:
1. Use a terminal and run the simulator to see logs if something fails:
   python simulator.py sample_map.json
2. Press S to toggle spawn mode and click two hubs to spawn a manual car.
3. Edit sample_map.json or create new maps in editor and load them in simulator
